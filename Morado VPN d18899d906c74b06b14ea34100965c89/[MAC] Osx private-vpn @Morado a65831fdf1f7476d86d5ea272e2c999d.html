AWS private-vpn @Morado

Go to download openvpn-client https://tunnelblick.net/downloads.html

Select Stable Version for Download and Doubleclick in dmg file:

https://tunnelblick.net/release/Tunnelblick_3.8.7a_build_5770.dmg

Install Tunnelblick:

DoubleClick in Tunnelblick icon and select open!

Insert your OSX credentials:

Tunnelblick begins the installation process.

Optional – You can use your personal VPN in my case I use ExpressVPN(Connect Please).

Configure Morado openvpn:

vpn.zip

Extract the vpn.zip in the Desktop

Open Tunnelblick and Drag and Drop the file called Morado.ovpn

Select users access (In my case I select Only-Me)

Insert you OSX credentials again.

Later, Select settings and configure this parameters :

Later, Click in Connect and Insert you Credentials for vpn access:

Request these credentials to administrator. ( devops-prod@Morado.co )

VPN Connected

Test access to AWS sending ping to address or Morado Private Url 🙂 Enjoy
